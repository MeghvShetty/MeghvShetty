## What I Work On

**AI & LLM Security**
Threat modelling of RAG pipelines, agentic AI systems, and GenAI 
integrations. Hands-on with prompt injection, model extraction, and 
adversarial input testing. Currently pursuing HTB Certified Offensive 
AI Expert (COAE).

**Application Security Engineering**
SAST/SCA pipeline integration via Semgrep and Checkmarx. DAST with 
Burp Suite. Secure SDLC design and developer security enablement 
across large engineering organisations.

**Security Tooling**
Built Apollo-24 in Go — a Security Impact Rating automation tool 
deployed in a regulated financial services environment. Reduced security 
triage overhead and enforced classification consistency across 
engineering teams.

**Governance & Standards**
ISO 42001 certified (AI Management Systems). Co-Authored group-wide AI 
Security Standard aligned to UK AI Code of Practice, NIST AI RMF, 
and ISO 27001.
